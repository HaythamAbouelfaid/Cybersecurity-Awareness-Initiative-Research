# Security Policy

## Reporting a Vulnerability
If you discover a cybersecurity issue or vulnerability related to the content in this project, please open an issue or contact the repository owner directly.

## Supported Content
This project contains:
- Research on cybersecurity awareness  
- Policy recommendations  
- Educational resources  

No active code or software vulnerabilities exist in this repository.

## Objective
This project aims to improve cybersecurity awareness for students in higher education and promote best practices for digital safety.
